# 📊 Analytics & Reporting

Módulo encargado de **suscribirse a todos los eventos** del sistema y generar dashboards gerenciales mediante procesos ETL.

## 🚀 Funcionalidades
- Suscripción a todos los eventos publicados en el Core.
- Procesos ETL para normalización de datos.
- Generación de dashboards para gerentes.
- KPIs sobre ventas, productos y usuarios.
